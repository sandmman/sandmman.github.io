Current Projects:

Remy (NES Emulator):

  • Follow along as I build an NES Emulator in Rust!

Asteroids:

  • A remake of the classic game in HTML5 and Javascript incorporating computational geometry principles. Using this project, I’ve been slowly fleshing out a simple game engine that will be used on future projects.

Soccer Stats Scraper:

  • A python scraper I made for my friend’s Economics thesis that scrapes the career stats of soccer players wikipedia pages.

Hearts:

  • Play/simulate a game of the card game hearts!

Snake:

  • A remake of snake in python using tkinter.

Heap Manager:

  • A simple implementation of a heap manager in c.

Shape Google

  • Implements various shape descriptor histograms applied to a database of point clouds and analyzes their ability to recognize input shape similarity.
